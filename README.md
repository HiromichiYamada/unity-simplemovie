unity-simplemovie
=================

Alternative to Handheld.PlayFullScreenMovie

Usage
=================

The source to open the movie at "/StreamingAssets/mymovie/dyson360eye.mp4"

```
SimpleMovie.PlayMovie("mymovie", "dyson360eye", "mp4");
```
